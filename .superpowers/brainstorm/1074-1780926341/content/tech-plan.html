关键设计决策

  • 不继承 base.html — 首页是独立浅色系风格，首页也没有用 base.html，教程页保持一致
  • 零 JS 逻辑 — 纯静态内容页，不需要交互。视频用 iframe 直接嵌入
  • 图片用 OSS 外链 — 你已上传阿里云，模板中直接用 URL 引用
  • Tailwind + Remix Icon CDN — 与 index.html 保持一致

不需要做的

  • 不需要登录鉴权（教程页对未登录用户也可访问）
  • 不需要后端 API、数据库改动
  • 不需要 JS 交互逻辑
